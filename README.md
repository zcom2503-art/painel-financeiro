# 📊 Painel Financeiro e Simulador

Este é um projeto prático focado na construção de aplicações web interativas. Trata-se de um painel financeiro dinâmico onde o usuário pode registrar entradas e saídas, calculando automaticamente o saldo final e gerando um extrato de operações em tempo real.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação da aplicação e formulários.
* **CSS3:** Estilização moderna inspirada em terminais financeiros (Dark Mode) e layout flexível (Flexbox).
* **JavaScript (Vanilla):** Lógica de programação para cálculos matemáticos, manipulação do DOM, formatação de moedas e atualização da interface.

## 🚀 Funcionalidades
* Adição de receitas (entradas) e despesas (saídas).
* Cálculo automático e dinâmico de saldo total.
* Extrato visual com identificação intuitiva por cores (Verde para entrada, Vermelho para saída).
